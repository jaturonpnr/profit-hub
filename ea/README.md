# ProfitHub Collector EA — คู่มือติดตั้ง

EA ตัวนี้ทำหน้าที่ **อ่าน** ประวัติดีล (closed deals) จากบัญชี MT5 แล้วส่งขึ้น Profit Hub API เท่านั้น — **ไม่มีการเปิด/ปิด/แก้ไขออเดอร์ใด ๆ ทั้งสิ้น**

## ทำงานร่วมกับ EA เทรดตัวอื่นได้อย่างปลอดภัย

- อ่านข้อมูลอย่างเดียว (read-only) — ไม่ส่งคำสั่งเทรดเด็ดขาด
- รันบนชาร์ตของตัวเอง แยกเธรดจาก EA เทรดบนชาร์ตอื่น
- ใช้ `OnTimer` ทุก 1–5 นาที (ไม่ใช่ `OnTick`) จึงแทบไม่กิน CPU
- `WebRequest` timeout สั้นแค่ 5 วินาที — ถ้าส่งไม่สำเร็จจะ "ปล่อยผ่าน" แล้วรอรอบถัดไป ไม่มี retry loop ที่จะค้างเทอร์มินัล
- ฝั่งเซิร์ฟเวอร์เป็น idempotent — ส่งซ้ำได้เสมอ ข้อมูลไม่หายและไม่ซ้ำ

## ขั้นตอนติดตั้ง

1. **คัดลอกไฟล์ EA**
   - ใน MT5 ไปที่ `File → Open Data Folder`
   - วางไฟล์ `ProfitHubCollector.mq5` ลงในโฟลเดอร์ `MQL5/Experts/`

2. **คอมไพล์**
   - เปิด MetaEditor (กด F4 ใน MT5)
   - เปิดไฟล์ `ProfitHubCollector.mq5` แล้วกด **Compile** (F7)
   - ต้องได้ **0 errors**

3. **อนุญาต WebRequest (สำคัญมาก — ถ้าข้ามขั้นนี้ EA จะส่งข้อมูลไม่ได้)**
   - ใน MT5 ไปที่ `Tools → Options → Expert Advisors`
   - ติ๊ก **Allow WebRequest for listed URL**
   - เพิ่ม URL ของ API เช่น `https://your-api.onrender.com` (ใส่เฉพาะ base URL ให้ตรงกับค่า `ApiUrl` ที่จะตั้งใน EA)

4. **เปิดชาร์ตว่าง 1 ชาร์ต**
   - เปิดชาร์ตใหม่ (คู่เงิน/timeframe อะไรก็ได้ — EA ไม่ได้ใช้ราคาจากชาร์ต)
   - แนะนำให้ใช้ชาร์ตที่ไม่มี EA เทรดรันอยู่ — **1 เทอร์มินัล ติดตั้ง EA ตัวนี้แค่ชาร์ตเดียว**

5. **ลาก EA ลงชาร์ต แล้วตั้งค่า**
   - `ApiUrl` — base URL ของ Profit Hub API (ตรงกับที่ whitelist ไว้ในข้อ 3)
   - `IngestKey` — คัดลอก **Ingest Key** ของบัญชีนี้จากหน้า **Accounts** บนเว็บ Profit Hub มาวาง
   - `IntervalSec` — ความถี่ในการส่ง (ค่าเริ่มต้น 120 วินาที, ขั้นต่ำ 60)
   - `BatchSize` — จำนวนดีลต่อ 1 request (ค่าเริ่มต้น 100, สูงสุด 1000 — เกินกว่านี้ backend จะปฏิเสธ)
   - `ForceBackfill` — ติ๊ก **ครั้งเดียว** เพื่อล้าง watermark แล้วส่งประวัติทั้งหมดใหม่ จากนั้น **untick แล้วแนบ EA ใหม่** (backend กันซ้ำ ส่งซ้ำไม่เบิ้ล) — ใช้เวลาข้อมูลหายหรืออยากดึงย้อนหลังใหม่

6. **เปิด Algo Trading**
   - กดปุ่ม **Algo Trading** บน toolbar ให้เป็นสีเขียว
   - มุมขวาบนของชาร์ตต้องเห็นไอคอน EA แบบไม่มีกากบาท

> ครั้งแรกที่แนบ EA จะส่งประวัติทั้งหมดขึ้นไป (backfill) จากนั้นจะส่งเฉพาะดีลใหม่ (delta) — ตำแหน่งล่าสุดถูกจำไว้ใน Global Variable ของเทอร์มินัล แม้ปิด/เปิด MT5 ใหม่ก็ไม่ส่งซ้ำโดยไม่จำเป็น

## เช็คลิสต์ตรวจสอบหลังติดตั้ง

- [ ] คอมไพล์ผ่านด้วย 0 errors
- [ ] แนบ EA ครั้งแรกแล้ว ประวัติเทรดทั้งหมดขึ้นบนเว็บ (ดูที่หน้า Trades)
- [ ] หน้า **Accounts** แสดง `Last Ingest` อัปเดตตามรอบ timer
- [ ] ทดลองตัดเน็ตระหว่างส่ง — ข้อมูลไม่หาย (รอบถัดไปส่งซ้ำให้เอง เพราะฝั่งเซิร์ฟเวอร์กันข้อมูลซ้ำ)
- [ ] แท็บ **Experts** ใน MT5 ไม่มี error ค้าง (ถ้าเห็น `push failed status=-1` แปลว่ายังไม่ได้ whitelist URL ตามข้อ 3)

## แก้ปัญหาเบื้องต้น

| อาการ | สาเหตุ/วิธีแก้ |
|---|---|
| `IngestKey is empty` ตอนแนบ EA | ยังไม่ได้ใส่ Ingest Key ในหน้าตั้งค่า EA |
| `push failed status=-1` | ยังไม่ได้ whitelist URL ใน Tools → Options → Expert Advisors |
| `push failed status=401` | Ingest Key ไม่ถูกต้อง — คัดลอกใหม่จากหน้า Accounts |
| ข้อมูลไม่ขึ้นเลย | เช็คว่า Algo Trading เปิดอยู่ (ปุ่มสีเขียว) และไอคอน EA บนชาร์ตไม่มีกากบาท |
